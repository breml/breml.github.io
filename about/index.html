What I do

I currently work as a freelance software architect and software engineer. I am specialized in software development, especially with the Go Programming Language, cloud infrastructure and DevOps. I am an experienced team and project leader with in depth knowledge in the fields of network, security, Linux engineering and cloud native solutions. I have strong record of developing and customizing solutions for monitoring and operating distributed systems.

I am a Go and open source enthusiast, I am keen about automated testing and linting and I have big interests in cloud native solutions, metrics, and log management. This includes solutions based on Elasticsearch, Logstash, Amazon Web Services (AWS), Ansible and Puppet. Unfortunately a lot of the glue code, we use to put these pieces together, is closed source.

Open Source

My story with open source software starts during my education, when I did my first steps with Linux, PHP and MySQL. My first open source contribution dates back in 1999 where I helped with a small fix to the solaris build of mod_ssl. Ever since I contributed to several open source projects in multiple programming languages like Go, Java, PHP, Python and Ruby. Some of my last recent contributions may be found on Github as PR and issues.
